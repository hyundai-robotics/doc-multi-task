# 2.1.5 task reset

task reset 명령문은 서브태스크를 강제로 소멸하는 역할을 수행합니다. 일반적으로 서브태스크 소멸은 해당 서브태스크 프로그램의 end 명령문 실행에 의해 자동으로 처리됩니다. 

```
task reset,sub=<서브태스크 번호>
```

|    **항목**    | 　　　　　　　　　　**내용**                                  |
| :----------: | ------------------------------------------------- |
| **서브태스크 번호** | 소멸을 원하는 서브태스크 번호를 지정(1\~7)                       |
|   **사용 예시**  | <p>task reset,sub=1</p><p>(서브태스크 1을 소멸합니다.)</p> |
