# 2.1.4 task stop

task stop 명령문은 서브태스크의 실행을 강제로 정지하는 역할을 수행합니다. 

```
task stop,sub=<서브태스크 번호>
```


|    **항목**    | 　　　　　　　　　　**내용**                                  |
| :----------: | ------------------------------------------------- |
| **서브태스크 번호** | 정지를 원하는 서브태스크 번호를 지정(1\~7)                       |
|   **사용 예시**  | <p>task stop,sub=1</p><p>(서브태스크 1의 실행을 정지합니다.)</p> |
